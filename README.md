# Civic App

A Next.js civic engagement app — community-powered voting guides, forums, and issue tracking for local elections.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

```bash
npm run build
npm start
```

## Features

- **Issues Feed** — Browse community-reported issues with map view, topic filters, and voting
- **Forums** — Upcoming and past voter forums with candidate Q&A
- **Voting Guide** — Personalized candidate recommendations based on your engagement
- **Citizen's Voter Guide** — Community-wide issue alignment and candidate scoring

## Tech Stack

- Next.js 14 (App Router)
- React 18
- TypeScript
